# afira

Developer tool for automating common tasks.

```
Usage: afira [options] [command]

Developer tool for automating common tasks.

Options:
  -V, --version         output the version number
  -h, --help            output usage information

Commands:
  accumulate [options]  Accumulates files with specific extensions into a single folder. Handy for backing up files.
  cordova:check         Run production checks for a specific cordova project. Must be run from within a cordova project.
```
